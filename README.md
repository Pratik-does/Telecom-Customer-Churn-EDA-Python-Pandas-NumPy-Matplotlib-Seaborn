A recruiter-ready exploratory data analysis project on telecom customer churn. The objective is to identify the strongest churn drivers and translate them into actionable retention insights using Python-based analysis and visualization.

---

## Project Snapshot

This project analyzes **7,043 telecom customers** and explores how customer demographics, contract type, tenure, billing behavior, and service adoption influence churn.

The analysis was performed as an end-to-end EDA workflow:
- data loading and inspection
- data cleaning
- feature transformation
- churn distribution analysis
- customer segmentation
- revenue impact analysis
- correlation analysis
- business recommendations
